개인정보처리방침

Privacy Policy · 최종 업데이트: 2026년 6월 15일

APIKeyManager(이하 "서비스")는 이용자의 개인정보를 중요시하며, 관련 법률을 준수합니다. 본 방침은 서비스 이용 시 수집되는 정보와 그 처리 방법을 설명합니다.

1. 수집하는 정보

APIKeyManager 앱은 다음 정보를 로컬 기기에만 저장합니다.

  • API 키 — 암호화된 형태로 로컬 vault 파일에 저장. 외부 서버로 전송되지 않습니다.
  • 브라우저 세션 쿠키 — 자동 로그인을 위해 로컬에 저장. 외부 전송 없음.
  • 실행 로그 — 앱 동작 기록. 로컬에만 저장됩니다.
본 앱은 어떠한 개인 식별 정보도 외부 서버로 전송하지 않습니다.

2. 정보 이용 목적

  • API 키의 안전한 암호화 저장 및 복원
  • 서비스 콘솔 자동 로그인 (세션 유지)
  • 개발 환경 .env 파일 자동 업데이트

3. 정보 보관 및 파기

모든 데이터는 사용자의 로컬 기기에 저장됩니다. 앱 폴더 내 vault/keys.vault 파일을 삭제하면 저장된 모든 API 키가 영구 삭제됩니다. Windows Credential Manager의 마스터 키(api_automator_v1)도 별도로 삭제할 수 있습니다.

4. 제3자 서비스

본 웹사이트는 다음 제3자 서비스를 사용할 수 있습니다.

  • Google AdSense — 광고 표시 목적. Google의 개인정보처리방침이 적용됩니다.
  • Google Analytics — 방문자 통계 목적 (익명 처리).

각 제3자 서비스의 개인정보처리방침은 해당 서비스 사이트에서 확인하세요.

5. 쿠키 정책

본 웹사이트는 방문자 통계 및 광고 서비스를 위해 쿠키를 사용할 수 있습니다. 브라우저 설정에서 쿠키를 거부할 수 있으나, 일부 기능이 제한될 수 있습니다.

6. 이용자의 권리

  • 개인정보 열람, 수정, 삭제를 요청할 권리
  • 로컬 저장 데이터는 앱 폴더 삭제로 직접 제거 가능
  • 관련 문의: 문의 페이지를 이용하세요

7. 방침 변경

개인정보처리방침이 변경될 경우 본 페이지에 게시하여 공지합니다. 정기적으로 확인하시기를 권장합니다.

8. 문의

개인정보 관련 문의사항은 문의 페이지를 통해 연락해 주세요.

Privacy Policy (English)

APIKeyManager stores all data locally on your device. No personal information is transmitted to external servers. API keys are encrypted using Fernet (AES-128-CBC + HMAC-SHA256). The master key is stored in Windows Credential Manager. This website may use Google AdSense for advertising and Google Analytics for anonymous traffic statistics. For any privacy-related inquiries, please visit our contact page.
