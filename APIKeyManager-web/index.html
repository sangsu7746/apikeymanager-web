주요 기능
Key Features
🔐

군사급 암호화 저장

AES-128-CBC + HMAC-SHA256 (Fernet) 방식으로 마스터 키는 Windows Credential Manager에 저장됩니다. 평문은 절대 디스크에 기록되지 않습니다.

🤖

멀티 서비스 지원

Anthropic Claude, OpenAI GPT, Google Gemini, GitHub Token, AWS IAM — 5개 주요 AI 서비스를 하나의 화면에서 관리합니다.

📋

원클릭 복사

저장된 키 탭에서 📋 복사 버튼 하나로 API 키를 클립보드에 복사합니다. 키 값이 화면에 노출되지 않아 보안이 유지됩니다.

⚡

자동 .env 업데이트

발급된 API 키를 프로젝트의 .env 파일에 자동으로 기록합니다. 개발 환경 설정 시간을 대폭 줄여줍니다.

🗂️

프로젝트별 분류

API 키에 프로젝트 이름을 붙여 구분 관리합니다. 같은 서비스의 키라도 프로젝트별로 독립적으로 저장됩니다.

🌙

다크 테마 GUI

GitHub Dark 팔레트 기반의 세련된 인터페이스. 개발자 친화적 디자인으로 장시간 사용해도 눈이 편안합니다.

사용 방법
How It Works — 3단계로 끝납니다
1

서비스 선택

관리할 AI 서비스를 선택합니다. Anthropic, OpenAI, Gemini, GitHub, AWS 중 원하는 서비스를 체크하세요.

2

프로젝트 이름 입력 후 자동화 시작

프로젝트 이름을 입력하고 자동화를 시작하면, 각 서비스 콘솔에서 API 키가 자동으로 발급됩니다.

3

📋 복사해서 바로 사용

발급된 키는 암호화되어 안전하게 저장됩니다. 필요할 때 복사 버튼 하나로 즉시 사용하세요.
